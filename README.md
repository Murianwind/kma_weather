# 기상청 스마트 날씨 (KMA Smart Weather)

기상청 단기/중기 예보와 에어코리아 미세먼지 데이터를 통합하여 제공하는 Home Assistant용 커스텀 컴포넌트입니다.

## 📋 사전 준비 사항
이 통합 구성요소를 사용하려면 [공공데이터포털](https://www.data.go.kr/) 계정이 필요하며, 다음 **3가지 서비스**를 신청하여 승인을 받아야 합니다. (인증키는 모두 동일하게 사용 가능합니다.)

1. **기상청_단기예보 조회서비스** (실시간 날씨 및 내일 예보)
2. **기상청_중기예보 조회서비스** (3일~10일 주간 예보)
3. **한국환경공단_에어코리아_대기오염정보** (미세먼지 및 대기질 등급)

> **주의:** API 신청 후 데이터가 활성화되기까지 최대 24시간이 소요될 수 있습니다. 반드시 **인코딩(Encoding)된 인증키**를 사용하세요.
