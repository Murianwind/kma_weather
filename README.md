# KMA Smart Weather for Home Assistant

기상청 API와 에어코리아 API를 결합한 스마트 날씨 컴포넌트입니다.

## 주요 기능
- **기상청 단기예보**: 격자 좌표(NX, NY) 자동 변환 및 3시간 주기 업데이트.
- **동적 위치 추적**: `zone.home` 뿐만 아니라 스마트폰의 `device_tracker` 위치를 실시간 반영.
- **에어코리아 통합**: 가장 가까운 측정소를 자동으로 찾아 미세먼지 정보 제공.
- **API 키 관리**: 인증키 만료 전(2년 기준) 남은 일수 알림 센서 제공.

## 설치 방법
1. HACS > Custom Repositories에 이 주소를 추가하세요.
2. `기상청 스마트 날씨`를 검색하여 설치합니다.
3. HA 설정 > 기기 및 서비스 > 통합구성요소 추가에서 API 키를 입력합니다.

## API 키 발급처
- [공공데이터포털 - 기상청 단기예보](https://www.data.go.kr/)
- [공공데이터포털 - 에어코리아 대기오염](https://www.data.go.kr/)
